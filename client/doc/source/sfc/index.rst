Описание компонентов Vue
===========

Однофайловые компоненты Vue (они же файлы /*.vue, или называемые сокращённо SFC, от Single File Component) — специальный формат файлов, позволяющий собрать в одном файле шаблон, логику и стилизацию компонента Vue.
В данном разделе документации будет подробно рассмотрены все используемые SFC.
